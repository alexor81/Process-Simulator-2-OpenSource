<ul>

</ul>

<ul>
<li><strong>Animation.ImageMove</strong> - move and rotate image.</li>

<li><strong>Item.Delay</strong> - copy one Item value to another on command with adjustable delay.</li>
<li><strong>Item.TimeLine</strong> - write values to Items at intervals.</li>

<li><strong>Pipeline.Pump</strong> - pump simulation.</li>
<li><strong>Pipeline.Valve</strong> - valve actuator simulation.</li>

<li><strong>Real.Comparator</strong> - comparison of two values.</li>
